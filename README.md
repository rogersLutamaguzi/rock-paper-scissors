
**ROCK PAPER SCISSORS GAME** 

I built this game because I wanted to see if I could actually make something interactive with JavaScript. It’s a classic Rock-Paper-Scissors setup where I coded a "computer" to play against me.

**WHY I BUILT THIS**

Since I’m still new to coding, my main goals were:

* Logic: I wanted to figure out the "if/else" math for who wins (like Rock beating Scissors).
* Saving my progress: I used localStorage so my score doesn't disappear if I refresh the page. This was the trickiest part for me to get right!
* Updating the screen: I learned how to make the text on the page change instantly based on my clicks.

**HOW I MADE IT WORK**

* I used Math.random() to let the computer "decide" its move.
* I wrote a function that compares my choice to the computer's choice.
* The page keeps a running tally of my wins, losses, and ties.
* I added a "Reset Score" button for when I want to start over from zero.

**HOW TO PLAY**

You can just open the index.html file in any browser to try it out.
Live Demo: rogerslutamaguzi.github.io/rock-paper-scissors

## What I struggled with

Honestly, at first, my score kept resetting to zero every time I hit refresh. Learning how to turn my score object into a string to save it, and then back into an object to read it, was a huge "lightbulb" moment for me.
